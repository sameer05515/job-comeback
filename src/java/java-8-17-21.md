# java 8 features.

## ✅ Java 8 — Most Important Features (with code)

---

### 1️⃣ Lambda Expressions (Functional Programming)

```java
// Before Java 8
Runnable r1 = new Runnable() {
    @Override
    public void run() {
        System.out.println("Hello");
    }
};

// Java 8
Runnable r2 = () -> System.out.println("Hello");
new Thread(r2).start();
```

---

### 2️⃣ Functional Interfaces

```java
@FunctionalInterface
interface MyFunc {
    void sayHello();
}

public class Test {
    public static void main(String[] args) {
        MyFunc f = () -> System.out.println("Hello Functional");
        f.sayHello();
    }
}
```

Common built-in ones:

```java
Predicate<Integer> p = x -> x > 10;
Function<Integer, String> f = x -> "Value: " + x;
Consumer<String> c = x -> System.out.println(x);
Supplier<Double> s = () -> Math.random();
```

---

### 3️⃣ Stream API (Most Asked in Interviews 🔥)

```java
List<Integer> list = Arrays.asList(1,2,3,4,5,6);

// filter + map + collect
List<Integer> result =
        list.stream()
            .filter(x -> x % 2 == 0)
            .map(x -> x * x)
            .collect(Collectors.toList());

System.out.println(result);
```

Parallel Stream:

```java
list.parallelStream()
    .forEach(System.out::println);
```

---

### 4️⃣ Method References (::)

```java
List<String> names = Arrays.asList("A", "B", "C");

names.forEach(System.out::println);
```

Types:

```java
ClassName::staticMethod
object::instanceMethod
ClassName::new
```

---

### 5️⃣ Default & Static Methods in Interface

```java
interface Vehicle {

    default void start() {
        System.out.println("Vehicle started");
    }

    static void stop() {
        System.out.println("Vehicle stopped");
    }
}

class Car implements Vehicle {}

public class Test {
    public static void main(String[] args) {
        Car c = new Car();
        c.start();
        Vehicle.stop();
    }
}
```

---

### 6️⃣ Optional Class (Null Safety)

```java
Optional<String> name = Optional.ofNullable(null);

name.ifPresent(System.out::println);
```

### 7️⃣ Date & Time API (java.time package)

Formatting:

---

### 8️⃣ forEach() Method

```java

```

---

### 9️⃣ Nashorn JavaScript Engine (Removed later)

# Java 21 New Features

## ✅ Java 21 Features (LTS — Latest 🔥)

---

## 1️⃣ Virtual Threads (Most Important 🚀)

Lightweight threads for high-concurrency apps.

```java
try (var executor = java.util.concurrent.Executors.newVirtualThreadPerTaskExecutor()) {

    for (int i = 0; i < 10; i++) {
        int num = i;
        executor.submit(() -> {
            System.out.println("Task: " + num + " -> " + Thread.currentThread());
        });
    }
}
```

Simple:

```java
Thread.startVirtualThread(() -> {
    System.out.println("Hello Virtual Thread");
});
```

Best for:

```
Microservices
APIs
I/O heavy apps
High concurrency systems
```

---

## 2️⃣ Structured Concurrency (Preview)

Manage multiple threads as one unit.

```java
import java.util.concurrent.StructuredTaskScope;

try (var scope = new StructuredTaskScope.ShutdownOnFailure()) {

    var user = scope.fork(() -> fetchUser());
    var order = scope.fork(() -> fetchOrder());

    scope.join();
    scope.throwIfFailed();

    System.out.println(user.result());
    System.out.println(order.result());
}
```

---

## 3️⃣ Record Patterns (Pattern Matching Upgrade)

```java
record Point(int x, int y) {}

static void print(Object obj) {
    if (obj instanceof Point(int x, int y)) {
        System.out.println(x + "," + y);
    }
}
```

Nested:

```java
if (obj instanceof Rectangle(Point(int x1, int y1), Point(int x2, int y2))) {
    System.out.println("Rectangle");
}
```

---

## 4️⃣ Pattern Matching for Switch (Final 🔥)

```java
static String format(Object obj) {
    return switch (obj) {
        case Integer i -> "Integer: " + i;
        case String s -> "String: " + s;
        case null -> "Null";
        default -> "Unknown";
    };
}
```

With guards:

```java
case String s when s.length() > 5 -> "Long String";
```

---

## 5️⃣ Sequenced Collections (New Interfaces)

New interfaces:

```
SequencedCollection
SequencedSet
SequencedMap
```

Example:

```java
SequencedCollection<String> list = new ArrayList<>();

list.addFirst("A");
list.addLast("B");

System.out.println(list.getFirst());
System.out.println(list.getLast());
```

Map:

```java
SequencedMap<Integer, String> map = new LinkedHashMap<>();

map.putFirst(1, "One");
map.putLast(2, "Two");
```

---

## 6️⃣ String Templates (Preview)

```java

```

Custom processor:

## 2️⃣ What is Record in Java 17?

Immutable data carrier class.

```java
public record User(Long id, String name) {}
```

```java

System.out.println(u.name());
```

Auto generates:

## 3️⃣ Record vs Normal Class?

```java
// Normal
class User {
    private final Long id;
    private final String name;

    public User(Long id, String name) {
        this.id = id;
        this.name = name;
    }

    public Long getId() { return id; }
    public String getName() { return name; }

```

```java
// Record
record User(Long id, String name) {}
```

## 4️⃣ Can we create mutable Record?

Yes (not recommended).

```java

```

## 5️⃣ What is Sealed Class?

Restrict inheritance.

```java
public sealed class Shape
        permits Circle, Rectangle {}

final class Rectangle extends Shape {}
```

---

## 6️⃣ Sealed vs Final vs Abstract

```
final     → no inheritance
sealed    → limited inheritance
abstract  → unrestricted inheritance
```

## 7️⃣ What is non-sealed?

Allows further extension.

```java
public non-sealed class Square extends Shape {}
```

---

## 8️⃣ Pattern Matching for instanceof?

Removes explicit casting.


## 9️⃣ Switch Expression in Java 17?

```java
    case 1 -> "Mon";
    case 2 -> "Tue";
    default -> "Other";

```


```text
I/O operations
Microservices
Database calls
API servers
High concurrency systems
```

# Java 8 Interview Questions

## ✅ Java 8 Interview Questions (Most Asked 🔥)

---

## 1️⃣ What are the main features of Java 8?

```text
Lambda Expressions
Functional Interfaces
Stream API
Method References
Optional Class
Default & Static Methods in Interface
Date & Time API (java.time)
forEach()
Nashorn Engine
```

---

## 2️⃣ What is Lambda Expression?

Anonymous function (no name, no class).

```java
Runnable r = () -> System.out.println("Hello");
new Thread(r).start();
```

Syntax:

```java
(parameters) -> expression
```

---

## 3️⃣ What is Functional Interface?

Interface with **only one abstract method**.

```java
@FunctionalInterface
interface MyFunc {
    void apply();
}
```

Usage:

```java
MyFunc f = () -> System.out.println("Run");
```

---

## 4️⃣ Built-in Functional Interfaces?

```java
Predicate<T>
Function<T,R>
Consumer<T>
Supplier<T>
UnaryOperator<T>
BinaryOperator<T>
```

Examples:

```java
Predicate<Integer> p = x -> x > 10;
Function<Integer, String> f = x -> "Val " + x;
Consumer<String> c = System.out::println;
Supplier<Double> s = Math::random;
```

---

## 5️⃣ What is Stream API?

Process collections in functional style.

```java
List<Integer> list = List.of(1,2,3,4);

list.stream()
    .filter(x -> x % 2 == 0)
    .map(x -> x * 2)
    .forEach(System.out::println);
```

---

## 6️⃣ Stream vs Collection?

```text
Collection → stores data
Stream → processes data
```

---

## 7️⃣ Intermediate vs Terminal Operations?

```text
Intermediate → filter, map, sorted
Terminal → collect, forEach, count
```

Example:

```java
list.stream()
    .filter(x -> x > 2)   // intermediate
    .count();             // terminal
```

---

## 8️⃣ What is Optional?

Avoid NullPointerException.

```java
Optional<String> name = Optional.ofNullable(null);

```

## 9️⃣ Optional Methods?


## 🔟 Default Methods in Interface?
